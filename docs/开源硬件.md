# gpioArduino
![arduino](images/arduino/ext_arduino_cog.png)  
## gpioArduino介绍
    通过连接Arduino，Arduino板卡能够读取传感器的模拟或数字输入信号，并将其转换为输出，例如激活电机，打开/关闭LED多种操作。

## 连接arduino
![arduino](images/arduino/scan_com_arduino.png)
	
	通过数据线与Arduino相连，点击gpioArduino插件。
