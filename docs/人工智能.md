## 人脸搜索
|指令     |    功能 |
| --- | --- |
|  ![face recognition *=300x50*](./images/创建人脸组.png) |  首先创建一个人脸组，为添加人脸到组做准备。 人脸组的名称可以自己定义，相同名称的人脸组只需要新建一次。  |
|  ![face recognition *=300x40*](./images/添加人脸到组.png) | 将人脸添加到提前建好的人脸组中，确保摄像头打开，并正对摄像头。第一个参数填写添加的人脸对应的姓名。第二个参数填写组名。在人脸组中添加过的人脸会被保存。    |
| ![face recognition *=300x50*](./images/搜索人脸.png)   |  在对应的人脸组里搜索摄像头前的人脸，确保摄像头打开，并正对摄像头 。 |
|  ![face recognition *=150x70*](./images/搜索完成事件.png)  |   当人脸搜索完成后，执行对应的事件。  |
| ![face recognition *=200x50*](./images/人脸搜索结果.png)|   返回人脸搜索后得到的姓名。  |

|参考程序 |    说明 |
| --- | --- |
| ![face recognition](./images/人脸搜索案例.png) |  <center>点击绿旗后，在group_1组中搜索人脸，当搜索完成后，如果搜索到的姓名等于自己的名字，小度音箱播报：欢迎回家。 </center> |
